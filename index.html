About Me

Full Stack Developer

A passionate and dedicated Full Stack Developer with a knack for creating dynamic and user-friendly web applications. With a strong foundation in both frontend and backend technologies, I excel in crafting responsive and interactive web interfaces using React.js, HTML, and CSS. I leverage modern CSS frameworks like Bootstrap and Tailwind CSS for efficient and stylish designs. On the backend, I develop robust server-side applications with Java and Spring Boot, and node.js as well as fast and lightweight solutions using Node.js and Express. My expertise extends to database management with SQL and MongoDB, ensuring flexible and efficient data storage. Additionally, I utilize tools like Git for version control, Docker for containerization, and Jenkins for CI/CD pipelines to automate the build, test, and deployment process.
